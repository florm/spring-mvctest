# 概要
このプロジェクトは、Spring MVC（3.2.0）の学習のために作成したサンプルです。

基本的なCRUDと、ファイルのアップロード、ダウンロード、画像の表示の機能を実装しています。

また、Spring MVCの3.2から導入された、MvcMockを使ったテストのサンプルもあわせて作成しています。
